== Description ==
Display your Gittip widget on your wordpress blog.

Features:

* Adds a Gittip widget to your available widgets.

== Installation ==

1. Upload the 'gittip-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
